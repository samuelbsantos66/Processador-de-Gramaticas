# Processador de Gramáticas — Python

Este repositório contém um processador completo de gramáticas, escrito em Python, capaz de:

✔ Ler uma gramática a partir de um arquivo
✔ Identificar o símbolo inicial
✔ Remover produções vazias (ε-produções)
✔ Remover produções unidade
✔ Remover símbolos inúteis
✔ Converter a gramática para Forma Normal de Chomsky (CNF)
✔ Derivar palavras passo a passo a partir da gramática na Forma Normal de Chomsky (CNF)
✔ Exibir todo o processo de derivação em formato detalhado
✔ Exportar gramáticas intermediárias para arquivos .txt

O sistema foi desenvolvido para fins didáticos, permitindo visualizar o funcionamento das transformações e derivação em linguagens formais.
